Detailed Description

signal Class

This is a class made to get the signal for 2 minutes. The signal comes in on pin D7 on an arduino. It's stored in the variable raw_data_buffer. The size of this variable is set so that 2 minutes and 4 seconds of the signal fit in. The fill method is used to fill the array. The get_signal method is then used to return a reference to the filled array so that other parts of the code can use this as well.
