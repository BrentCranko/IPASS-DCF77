Detailed Description

decoder Class

This class is made to interpret the data stored in the raw_data_buffer. It consists of a buffer for the complete signal, a decoded buffer that holds 59 spaces and a marker variable for the start and end of the signal. The class decodes the given raw data and puts it in an array. It's also the place where parity checks are made to see if the signal is correct.
